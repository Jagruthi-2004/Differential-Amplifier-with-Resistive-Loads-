# Differential-Amplifier-with-Resistive-Loads-
CMOS Differential Amplifier with resistive loads designed and simulated using Electric VLSI EDA tool and LTspice | 300nm MOSIS process | Schematic, Layout, Post-layout verification &amp; CMRR analysis
A full CMOS design flow implementation of a differential amplifier with resistive loads using MOSFET transistors, built as part of an analog VLSI design course under L&T Edutech.
Tech Stack & Tools

Electric VLSI EDA — Schematic & Layout design
LTspice — SPICE simulation
MOSIS C5 300nm CMOS Process
 Design Specs

Supply: VDD = 5V, VSS = 0V
5-transistor topology (2 PMOS loads + 2 NMOS diff pair + 1 NMOS tail)
Tail bias voltage: 0.9V
